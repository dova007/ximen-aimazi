## 修辞使用规则（按题材分级）

比喻和修辞不是绝对禁止，而是**按题材和场景分级使用**：

| 题材/场景 | 比喻限制 | 说明 |
|-----------|---------|------|
| 热血爽文风（默认） | 限制使用 | 允许功能性比喻（服务于动作/氛围），禁止装饰性比喻（纯为好看） |
| 古风仙侠风 | 适度放宽 | 允许意境类比喻（灵气如丝、剑气纵横），但每章不超过2处 |
| 都市现实风 | 严格限制 | 禁止比喻，用具体事实替代 |
| 悬疑惊悚风 | 严格限制 | 禁止比喻，用具体感官细节制造氛围 |
| 轻松日常风 | 适度放宽 | 允许幽默类比喻，但避免堆砌 |
| 擦边风格C（文艺唯美型） | 允许 | 允许自然意象比喻，这是该风格的核心特征 |

**装饰性比喻**（禁止）：为美化文字而写的比喻，删掉不影响理解。
**功能性比喻**（允许）：帮助读者理解动作、氛围、力量等级的比喻。

```
❌ 装饰性：她的眼睛如同星辰般明亮（删掉不影响理解）
✅ 功能性：他的拳风像闷雷一样炸开（帮助理解力量感）
```

### 1. 比喻词使用频率

| 词汇 | 限制 |
|------|------|
| 像、就像、好像、如同、仿佛、宛如、犹如、好似 | 每章不超过1处，且必须是功能性比喻 |

### 2. 弱化副词（频率限制）

以下词汇不是绝对禁止，但**每1000字不超过2个**，超过即视为AI签名：

```
微微、淡淡、缓缓、轻轻、不禁、不由自主
```

### 3. 直白心理描写（限制使用）

优先通过行为展示心理状态。以下词汇**每章不超过1处**：

```
紧张、焦急、愤怒、恐惧、兴奋、失落
```

正确做法：用动作/生理反应展示心理
```
❌ 他很紧张 → ✅ 他手心全是汗
❌ 她很焦急 → ✅ 她眼眶撑开了些
❌ 他感到愤怒 → ✅ 他没出声
```

### 4. 意义膨胀

AI喜欢把小事说成大事：

```
❌ "意义深远" → 写具体后果
❌ "前所未有" → 给出对比参照
❌ "可谓" → 删掉
```

### 5. 万能结论

章节结尾出现以下句式 = AI入侵：

```
❌ "未来可期" → 用未解决的紧张感结尾
❌ "前途无量" → 删
❌ "充满希望" → 写具体的下一步动作
❌ "他终于明白了……" → 删
❌ "这一夜，注定无人入眠" → 用动作收束
❌ "他不知道的是，更大的风暴即将来临" → 删
```

### 6. 论文体结构

小说中出现以下开头句 = AI痕迹：

```
"不难看出""由此可见""事实上""综上所述"
"于是乎""与此同时""从而""因而""诚然"
```

### 7. 三连排比癖

AI喜欢把事情凑成三个以显"完整"：

```
❌ 这次突破带来了创新、进步、变革
✅ 这次突破带来了进步

❌ 他冷静、沉着、有条不紊
✅ 他很冷静
```

## 中文去欧化

### 拒绝"很"和"非常"

```
❌ 他很高。她很美。这很重要。
✅ 他身材高大。她容貌出众。这事关成败。
```

### 拒绝"被"字滥用

```
❌ 他被激怒了。他被打败了。
✅ 他火了。他输了。
```

### 拒绝弱动词+名词

```
❌ 作出决定、进行分析、进行处理
✅ 决定、分析、处理
```

### 拒绝翻译腔

```
❌ 名为……的……
❌ 取而代之的是……
❌ 令人……
❌ 最……之一
❌ 如此……以至于……
```

## 文风例外说明

当激活特定文风时，以下规则可以放宽：

| 激活文风 | 放宽规则 | 限制条件 |
|---------|---------|---------|
| 古风仙侠风（文风C） | 允许四字词语、对偶、意境比喻 | 每章比喻不超过2处，禁止现代网络热梗 |
| 擦边风格C（文艺唯美型） | 允许自然意象比喻 | 仅限擦边情节中使用 |
| 轻松日常风（文风E） | 允许口语化比喻和幽默表达 | 禁止堆砌段子 |
| 悬疑惊悚风（文风D） | 允许氛围类比喻 | 禁止解释性比喻，必须服务于悬念 |

**铁律**：无论什么文风，以下规则不可放宽：
- 章末禁止总结性感悟/升华式感叹/哲理式收尾
- 段落不超过4行
- 禁止论文体结构（"不难看出""由此可见""综上所述"）

## 禁语表（频率限制版）

以下词汇不是绝对禁止，而是**按频率控制**。合理使用是有效的，滥用才会产生AI味：

### 结构套路类

| 词汇 | 频率限制 |
|------|---------|
| 名为、引以为傲、取而代之 | 每卷不超过1次 |
| 令人……、不容置疑、不容拒绝 | 每章不超过1次 |
| 就是现在、他动了 | 每章不超过1次（战斗场景除外） |

### 情绪状态类

| 词汇 | 频率限制 |
|------|---------|
| 一丝、一种、带着一丝、不易察觉 | 每章不超过2次 |
| 指节泛白、嘴角勾起弧度 | 每章不超过1次 |
| 笑容凝固、空气凝固 | 每章不超过1次 |
| 沉默了、眼神空洞 | 每章不超过1次 |

### 烂俗意象类

| 词汇 | 频率限制 |
|------|---------|
| 涟漪、波澜、心湖、深渊 | 每章不超过1次 |
| 四肢百骸、坠入、溺水之人 | 每章不超过1次 |

### 装逼文青类

| 词汇 | 频率限制 |
|------|---------|
| 张力、共鸣、精神共鸣、意味 | 每章不超过1次 |
| 狡黠、欲望、诗意、沉沦 | 每章不超过1次 |

### AI腔词汇

| 词汇 | 频率限制 |
|------|---------|
| 生理性、虔诚、骨节、恼怒、狰狞 | 每章不超过1次 |
| "他的语气"、"她的语气" | 每章不超过2次 |

## 擦边描写与去AI味规则优先级

当擦边描写规范（EDGE-TEMPLATE.md）与去AI味规则冲突时，按以下优先级处理：

| 优先级 | 来源 | 说明 |
|--------|------|------|
| 1（最高） | 用户本次明确指示 | 用户临时覆盖的规则 |
| 2 | 擦边风格C（文艺唯美型） | 允许在擦边情节中使用比喻 |
| 3 | 当前激活文风的例外规则 | 见上文"文风例外说明" |
| 4 | 去AI味基础规则 | 默认规则，无例外时适用 |

**冲突解决示例**：
- 擦边风格A要求"用心理活动制造张力（心跳加速、脸红）"，去AI味禁止"直白心理描写"
  → 解决：用生理反应替代（"他喉结动了一下"而非"他心跳加速"）
- 擦边风格C要求"用自然意象比喻"，去AI味禁止比喻
  → 解决：风格C为例外，允许在擦边情节中使用比喻

## 快速检查表

### 必检项

- [ ] 比喻词每章不超过1处，且为功能性比喻
- [ ] 弱化副词每1000字不超过2个
- [ ] 直白心理描写每章不超过1处
- [ ] 禁语表词汇未超频使用（见上文频率限制）
- [ ] "很"/"非常"未滥用
- [ ] 段落不超过4行
- [ ] 章尾无总结/升华/预告
- [ ] 当前文风的例外规则已正确应用

### 常见错误速查

| 错误 | 正确 |
|-----|------|
| 他很紧张 | 他手心全是汗 |
| 像一把刀 | 直接描述话语内容 |
| 他动了 | 具体写他做了什么 |
| 嘴角勾起 | 嘴角动了 |
| 空气凝固 | 没人说话 |
| 沉默了 | 过了几秒 |
| 不仅...而且... | ...和... |
| 希望这有帮助 | 删掉 |
