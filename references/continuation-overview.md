# 续写流程概览

> 本文件在 Continuation_Mode 激活时加载，提供续写准备阶段（CP1-CP3）的完整规范。
> 续写引擎详见 `continuation-engine.md`，风格指纹比对详见 `style-fingerprint-guide.md`，交互决策点详见 `continuation-interaction.md`。

---

## CP1-CP3 续写准备阶段

当 Continuation_Mode 激活时，在进入 Phase 7 之前须完成以下三个准备阶段：

| CP | 名称 | 目标 | 产出物 |
|----|------|------|--------|
| CP1 | 反向解析 | 从已有正文提取作品DNA、风格指纹、人物状态、伏笔状态、世界观规则 | `设定/作品DNA.md`、`设定/风格指纹.md`、角色卡、伏笔表、世界观文件、`.learnings/`回填 |
| CP2 | 卡点诊断与续写方案 | 诊断卡点类型，给出三条续写路线和3-10章节拍 | `设定/续写诊断.md`、`设定/续写方案.md` |
| CP3 | 不可改动清单确认 | 与用户确认不可推翻的设定清单（强门控） | `设定/不可改动清单.md`（Frozen_Canon 区非空） |

### 续写场景类型

| 编号 | 场景 | 触发词 |
|------|------|--------|
| A | 卡文诊断 | "卡文""写不下去了" |
| B | 外稿续写 | "续写""接着写""帮我续" + 提供正文>2000字 |
| C | 风格漂移修正 | "风格漂移""味道变了" |
| D | 旧坑回填 | "旧坑""填坑" |
| E | 跨作者接力 | "跨作者""接力" |

### 流程衔接

CP3 完成后跳过 Phase 1-6，直接进入 Phase 7。续写流程复用 Phase 7-12 的批次细纲与正文生成机制，不另起一套。

## 续写中的文风F（对比学习）激活

续写流程跳过 Phase 4 不等于不能使用文风F。以下三种调用方式均可在续写中途激活文风F：

| 调用方式 | 触发词/条件 | 何时可用 | 效果 |
|---------|------------|---------|------|
| **CP阶段贴入样本** | 在 CP1~CP3 期间贴入对比样本（AI原文 + 用户修改文） | CP准备阶段 | AI执行10维对比分析 → 写入 `memory/style_contrast_guide.md` → 更新 `memory/project_style.md` 中 `对比文风激活：是` → Phase 10/12 自动强制读取 |
| **Phase 10/12 中途调用** | 用户说"用F文风"/"激活对比文风"/提供对比样本 | 正文生成/润色任一阶段 | 中断当前Phase → 执行对比分析 → 更新 rule → 回流到当前Phase继续，文风F规则即时生效 |
| **已有guide直接激活** | `memory/style_contrast_guide.md` 和 `memory/project_style.md` 已存在 | 任意时刻 | 直接生效，无需重复提供样本。Phase 10/12 读取 `memory/project_style.md` 时发现 `对比文风激活：是` 即自动加载对比规则 |

> **样本要求**：提供一段AI原文（不想要的文风） + 一段用户修改文（想要的文风），AI 按 `references/style-contrast-analysis.md` 的10个维度自动分析差异。仅提供一段时，默认与 `memory/style_contrast_guide.md` 中预置12条通用规则比对。

## 交互模式

续写流程复用全流程交互模式（讨论模式/快速模式），与主流程统一。续写流程包含12个核心交互决策点（CP-①~CP-⑫）和1个可选文风F激活点（CP-⑦½），覆盖提取审核→诊断讨论→路线选择→节拍审核→锁定确认→文风对比激活→衔接确认→首章方向→试写讨论→情节点确认→审核讨论全流程。

### 续写决策点索引

| 编号 | 名称 | 所属阶段 | 快速模式 |
|------|------|----------|---------|
| CP-① | CP1 提取结果审核 | CP1 反向解析 | 跳过 |
| CP-② | CP1 关键设定修正 | CP1 反向解析 | 跳过 |
| CP-③ | CP2 卡点诊断讨论 | CP2 卡点诊断 | 跳过 |
| CP-④ | CP2 路线选择与调整 | CP2 卡点诊断 | **必停** |
| CP-⑤ | CP2 节拍审核 | CP2 卡点诊断 | 跳过 |
| CP-⑥ | CP3 锁定理由展示 | CP3 不可改动清单 | **必停** |
| CP-⑦ | CP3 补充遗漏条目 | CP3 不可改动清单 | 跳过 |
| CP-⑦½ | 文风F激活（可选） | CP3 ↔ Phase 7 过渡 | 跳过 |
| CP-⑧ | 衔接点确认 | Phase 7 细纲规划 | 跳过 |
| CP-⑨ | 首章方向选择 | Phase 8 生成细纲 | 跳过 |
| CP-⑩ | Sample Chapter 讨论 | Phase 10 正文 | **必停** |
| CP-⑪ | 每章关键情节点确认 | Phase 10 正文 | **必停(首章)** |
| CP-⑫ | 审核结果讨论 | Phase 12 润色评分 | 跳过(低于门槛停) |

### 快速模式 4 个必停点

| 必停点 | 理由 |
|--------|------|
| CP-④ 路线选择 | 续写方向决定后续质量，不可代劳 |
| CP-⑥ 锁定确认 | Frozen_Canon 一旦冻结影响全程，必须用户拍板 |
| CP-⑩ Sample Chapter | 试写定调，确认风格一致性 |
| CP-⑪ 首章情节点 | 首章定调，必须确认方向正确 |

## 风格指纹比对（9项指标）

| 指标 | 漂移阈值 |
|------|---------|
| 平均句长 | ±20% |
| 句长分布区间 | ±20% |
| 平均段长 | ±25% |
| 对话占比 | ±15% |
| 高频意象前20 | 重合度 ≥ 60% |
| 高频动词前20 | 重合度 ≥ 80% |
| 句式偏好 | ±20% |
| 动作-对话-情绪比例 | ±20% |
| 原作回避表达倾向 | 0（不得出现） |

3项以上指标超阈值 → 阻断润色，用户决定"重写/调整阈值/接受偏离"。Phase 12 额外执行风格回拉规则。
